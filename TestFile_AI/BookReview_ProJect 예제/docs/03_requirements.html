Software Requirements Specification (SRS)
버전 1.0 | 작성일: 2025.08.03 | 최종 수정일: 2025.08.21

📊 프로젝트 개요

🎯 프로젝트 목적

BookReview는 독서를 사랑하는 사용자들이 읽은 책에 대한 리뷰를 작성하고 공유할 수 있는 웹 기반 플랫폼입니다. 사용자들은 책에 대한 평점과 상세한 리뷰를 작성하여 다른 독서가들과 의견을 나누고, 새로운 책을 발견할 수 있는 커뮤니티 공간을 제공합니다.

🌟 핵심 가치

  • 📚 독서 문화 확산 및 책 추천 플랫폼
  • 🤝 독서가들 간의 소통과 지식 공유
  • ⭐ 신뢰할 수 있는 도서 평점 시스템
  • 🔍 개인화된 독서 취향 분석

👥 이해관계자 분석

이해관계자 역할 주요 관심사 영향도
일반 독서가 주 사용자 편리한 리뷰 작성, 신뢰할 수 있는 추천 높음
도서 애호가 활성 사용자 깊이 있는 리뷰 작성, 전문적 토론 높음
출판사 간접 이해관계자 도서 홍보, 독자 피드백 수집 중간
개발팀 시스템 운영자 안정적 서비스 운영, 사용자 만족도 높음
서점 운영자 비즈니스 파트너 도서 판매 증진, 고객 유입 낮음

📖 사용자 스토리

📚 독서가로서
읽은 책에 대한 솔직한 평가와 리뷰를 작성하여 다른 독서가들과 경험을 공유하고 싶다.
🔍 책을 찾는 사람으로서
다양한 도서에 대한 실제 독자들의 평점과 리뷰를 보고 읽을 책을 결정하고 싶다.
📊 독서 기록을 관리하는 사람으로서
내가 읽은 책들과 작성한 리뷰들을 한 곳에서 체계적으로 관리하고 싶다.
🎯 취향을 분석하려는 사람으로서
내가 선호하는 장르, 평균 평점 등의 독서 패턴을 분석하여 새로운 책을 발견하고 싶다.

⚙️ 기능 요구사항

FR-001
높음
사용자 회원 관리
사용자는 회원가입, 로그인, 로그아웃을 통해 개인 계정을 관리할 수 있어야 합니다. 회원가입 시 이메일 중복 확인과 비밀번호 강도 검증이 이루어져야 합니다.

인수 기준

  • 유효한 이메일 형식으로 회원가입 가능
  • 이메일 중복 시 적절한 오류 메시지 표시
  • 비밀번호는 8자 이상, 영문+숫자 조합
  • 로그인 상태가 브라우저에 유지
  • 로그아웃 시 세션 완전 종료
FR-002
높음
도서 목록 관리
시스템에 등록된 도서들을 검색, 필터링, 정렬하여 조회할 수 있어야 합니다. 도서 정보에는 제목, 저자, 출판사, 장르, 출간일, 설명이 포함됩니다.

인수 기준

  • 제목과 저자로 실시간 검색 가능
  • 장르별 필터링 기능
  • 최신순, 평점순, 이름순 정렬
  • 페이지네이션 (12개씩 표시)
  • 그리드/리스트 뷰 전환
FR-003
높음
리뷰 작성 및 관리
로그인한 사용자는 도서에 대한 리뷰를 작성, 수정, 삭제할 수 있어야 합니다. 리뷰에는 1-5점 별점, 리뷰 내용, 추천 여부가 포함됩니다.

인수 기준

  • 한 도서당 사용자 1개 리뷰만 작성 가능
  • 1-5점 별점 필수 입력
  • 리뷰 내용 최소 10자 이상
  • 추천/비추천 선택 가능
  • 자신의 리뷰만 수정/삭제 가능
FR-004
중간
도서 상세 정보
개별 도서의 상세 정보와 해당 도서의 모든 리뷰를 조회할 수 있어야 합니다. 도서의 평균 평점과 총 리뷰 수도 함께 표시됩니다.

인수 기준

  • 도서 기본 정보 전체 표시
  • 평균 평점과 총 리뷰 수 표시
  • 리뷰 목록 (최신순/평점순 정렬)
  • 리뷰 더 보기 기능 (5개씩 로딩)
  • 해당 도서 리뷰 작성 버튼
FR-005
중간
개인 리뷰 관리
사용자는 자신이 작성한 모든 리뷰를 한 곳에서 조회하고 관리할 수 있어야 합니다. 개인 독서 통계도 함께 제공됩니다.

인수 기준

  • 작성한 리뷰 전체 목록 표시
  • 총 리뷰 수, 평균 평점 통계
  • 추천한 책 수, 선호 장르 분석
  • 리뷰 정렬 (최신순, 평점순)
  • 개별 리뷰 수정/삭제 기능
FR-006
낮음
도서 추가/수정
로그인한 사용자는 시스템에 새로운 도서를 추가할 수 있으며, 자신이 추가한 도서는 수정/삭제할 수 있어야 합니다.

인수 기준

  • 제목, 저자는 필수 입력 항목
  • 출판사, 장르, 출간일은 선택 항목
  • 중복 도서 등록 방지 (제목+저자 기준)
  • 자신이 추가한 도서만 수정/삭제 가능
  • 도서 삭제 시 관련 리뷰도 함께 삭제

⚠️ 제약사항 및 가정

🚫 제약사항

  • 기술적 제약: 바닐라 JavaScript만 사용 (프레임워크 사용 금지)
  • 저장소 제약: localStorage 사용 (실제 데이터베이스 없음)
  • 시간적 제약: 개발 기간 28일 이내 완료
  • 예산적 제약: 무료 도구 및 리소스만 사용

💭 가정사항

  • 사용자 환경: 모던 웹 브라우저 사용 (IE 지원 불요)
  • 데이터 규모: 초기 100개 도서, 500개 리뷰 수준
  • 사용자 수: 동시 접속자 100명 이하
  • 언어 지원: 한국어만 지원 (국제화 불필요)
