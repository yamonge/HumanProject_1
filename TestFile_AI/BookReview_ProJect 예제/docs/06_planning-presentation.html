🎯 프로젝트 개요

📚 프로젝트 목적

독서를 사랑하는 사용자들이 읽은 책에 대한 리뷰를 작성하고 공유할 수 있는 웹 기반 플랫폼을 개발하여 독서 문화 확산에 기여

  • 🌟신뢰할 수 있는 도서 평점 시스템
  • 🤝독서가들 간의 소통 공간
  • 🔍개인화된 독서 취향 분석

👥 타겟 사용자

  • 📖일반 독서가: 책 추천이 필요한 사람들
  • ✍️리뷰어: 독서 후기를 공유하고 싶은 사람들
  • 📊독서 애호가: 체계적인 독서 관리가 필요한 사람들
  • 🎓학생/연구자: 학술서적 정보가 필요한 사람들

⚠️ 위험 관리

🔥 높은 위험도

일정 지연 가능성: 기능 복잡도가 예상보다 높을 수 있음

대응 방안: 주요 기능 우선 개발, 부가 기능은 2단계로 연기

⚡ 중간 위험도

브라우저 호환성 문제: 다양한 브라우저에서 테스트 필요

대응 방안: 주요 브라우저별 테스트 계획 수립, Polyfill 사용

📱 중간 위험도

모바일 최적화: 다양한 디바이스 크기 대응

대응 방안: Mobile-First 접근, 미디어 쿼리 적극 활용

✅ 낮은 위험도

기술적 난이도: 바닐라 JavaScript로 구현 가능한 수준

대응 방안: 단계적 개발, 충분한 테스트 시간 확보

🛡️ 위험 완화 전략

  • ✅ 애자일 개발: 짧은 스프린트로 빠른 피드백 수집
