🛠️ 기술적 도전과 해결

💾 localStorage 데이터 관리
관계형 데이터베이스의 JOIN 기능 없이 사용자, 도서, 리뷰 간의 관계를 효율적으로 관리해야 했습니다.
해결 방법:
UUID를 활용한 외래키 관계 구현, JavaScript의 filter()와 map() 메서드를 활용한 JOIN 연산 시뮬레이션, 데이터 무결성을 위한 참조 검사 로직 구현
🔍 실시간 검색 최적화
사용자가 타이핑할 때마다 즉시 검색 결과를 보여주되, 성능상 문제가 없도록 해야 했습니다.
해결 방법:
Debouncing 기법 적용 (300ms 지연), 대소문자 무시 검색, 부분 문자열 매칭, 검색 결과 캐싱을 통한 성능 최적화
📱 반응형 디자인 구현
다양한 디바이스 크기에서 일관된 사용자 경험을 제공하면서도 터치 인터페이스를 고려해야 했습니다.
해결 방법:
CSS Grid와 Flexbox 조합, 상대 단위(rem, %) 사용, 터치 타겟 크기 44px 이상 확보, 모바일 퍼스트 미디어 쿼리 적용
🔄 상태 동기화
여러 페이지 간 데이터 상태 동기화와 새로고침 시 상태 유지가 어려웠습니다.
해결 방법:
CustomEvent를 활용한 페이지 간 통신, localStorage 변경 감지, 페이지 로드 시 상태 복원 로직 구현
