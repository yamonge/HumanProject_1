2 HTML 구조 설계하기

HTML은 웹페이지의 뼈대입니다. 마치 집을 지을 때 기둥과 벽을 먼저 세우는 것과 같습니다.

📄 index.html (메인 페이지) 기본 구조

<!DOCTYPE html>
<html lang="ko">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>BookReview - 나만의 북 리뷰</title>
    <link rel="stylesheet" href="css/style.css">
</head>
<body>
    <header>
        <h1>📚 BookReview</h1>
        <nav>
            <a href="index.html">홈</a>
            <a href="book_add.html">책 추가</a>
            <a href="login.html">로그인</a>
        </nav>
    </header>
    
    <main>
        <section id="book-list">
            <h2>📖 등록된 책들</h2>
            <div id="books-container">
                <!-- JavaScript로 책 목록이 여기에 추가됩니다 -->
            </div>
        </section>
        
        <section id="recent-reviews">
            <h2>⭐ 최신 리뷰</h2>
            <div id="reviews-container">
                <!-- JavaScript로 리뷰 목록이 여기에 추가됩니다 -->
            </div>
        </section>
    </main>
    
    <script src="js/data.js"></script>
    <script src="js/script.js"></script>
</body>
</html>
💡 초보자 팁:
  • <header>: 웹사이트 상단 (제목, 메뉴)
  • <main>: 주요 내용이 들어가는 부분
  • <section>: 내용을 구역별로 나누는 것
  • id="": JavaScript에서 찾기 위한 이름표

✅ HTML 구조 체크리스트

  • index.html 메인 페이지 완성
  • login.html 로그인 페이지 완성

3 CSS 스타일링하기

CSS는 웹페이지를 예쁘게 꾸미는 코드입니다. 마치 집을 지은 후에 페인트칠하고 가구를 배치하는 것과 같습니다.

/* 전체 페이지 기본 설정 */

    max-width: 1200px;
    margin: 0 auto;

    margin: 30px 0;

    box-shadow: 0 5px 15px rgba(0,0,0,0.1);

/* 반응형 디자인 - 휴대폰에서 잘 보이게 */

    header h1 {
        font-size: 2em;
    }
    
    nav a {
        display: block;
        margin: 5px 0;
    }
    
    main {
        padding: 20px 10px;
    }
}
💡 CSS 초보자 가이드:
  • 선택자: 어떤 부분을 꾸밀지 선택 (header, .클래스명, #아이디명)
  • 속성: 무엇을 바꿀지 (color, font-size, background)
  • 값: 어떻게 바꿀지 (red, 20px, center)
  • @media: 화면 크기별로 다른 스타일 적용

4 JavaScript 기능 구현하기

JavaScript는 웹페이지가 동작하게 만드는 코드입니다. 마치 집에 전기를 연결해서 불이 켜지고 가전제품이 작동하게 하는 것과 같습니다.

⚡ js/data.js - 데이터 관리 함수들

// localStorage에서 데이터 가져오기
function getUsers() {
    const users = localStorage.getItem('bookReviewUsers');
    return users ? JSON.parse(users) : [];
}

function getBooks() {
    const books = localStorage.getItem('bookReviewBooks');
    return books ? JSON.parse(books) : [];
}

function getReviews() {
    const reviews = localStorage.getItem('bookReviewReviews');
    return reviews ? JSON.parse(reviews) : [];
}

// 데이터 저장하기
function saveUsers(users) {
    localStorage.setItem('bookReviewUsers', JSON.stringify(users));
}

function saveBooks(books) {
    localStorage.setItem('bookReviewBooks', JSON.stringify(books));
}

function saveReviews(reviews) {
    localStorage.setItem('bookReviewReviews', JSON.stringify(reviews));
}

// 새로운 사용자 추가
function addUser(name, email, password) {
    const users = getUsers();
    const newUser = {
        id: Date.now(), // 간단한 ID 생성
        name: name,
        email: email,
        password: password,
        joinDate: new Date().toISOString()
    };
    users.push(newUser);
    saveUsers(users);
    return newUser;
}

// 로그인 확인
function loginUser(email, password) {
    const users = getUsers();
    return users.find(user => user.email === email && user.password === password);
}

// 책 목록 표시
function displayBooks() {
    const books = getBooks();
    const container = document.getElementById('books-container');
    
    if (books.length === 0) {
        container.innerHTML = '<p>등록된 책이 없습니다. 첫 번째 책을 추가해보세요!</p>';
        return;
    }
    
    let html = '';
    books.forEach(book => {
        html += `
            <div class="book-card">
                <h3>${book.title}</h3>
                <p>저자: ${book.author}</p>
                <p>장르: ${book.genre}</p>
                <button onclick="viewBook(${book.id})">상세보기</button>
            </div>
        `;
    });
    
    container.innerHTML = html;
}

// 로그인 페이지 초기화
function initLoginPage() {
    const loginForm = document.getElementById('loginForm');
    if (loginForm) {
        loginForm.addEventListener('submit', handleLogin);
    }
}

// 로그인 처리
function handleLogin(event) {
    event.preventDefault(); // 폼 제출 기본 동작 방지
    
    const email = document.getElementById('email').value;
    const password = document.getElementById('password').value;
    
    const user = loginUser(email, password);
    
    if (user) {
        alert('로그인 성공!');
        localStorage.setItem('currentUser', JSON.stringify(user));
        window.location.href = 'index.html';
    } else {
        alert('이메일 또는 비밀번호가 틀렸습니다.');
    }
}
💡 JavaScript 초보자 가이드:
  • function: 특정 작업을 수행하는 코드 묶음
  • document.getElementById: HTML에서 특정 요소 찾기
  • addEventListener: 클릭, 입력 등의 사용자 동작 감지
  • localStorage: 브라우저에 데이터 저장하기

6 테스트 및 디버깅

테스트는 만든 기능이 제대로 작동하는지 확인하는 과정입니다. 마치 요리를 다 한 후에 간을 맞춰보는 것과 같습니다.

🐛 자주 발생하는 에러들과 해결법

// 에러 1: 요소를 찾을 수 없음
// ❌ 틀린 코드
document.getElementById('book-list').innerHTML = '내용';
// → book-list라는 id가 HTML에 없을 때 발생

// ✅ 올바른 코드
const bookList = document.getElementById('book-list');
if (bookList) {
    bookList.innerHTML = '내용';
} else {
    console.log('book-list 요소를 찾을 수 없습니다');
}

// 에러 2: localStorage 데이터가 없을 때
// ❌ 틀린 코드
const users = JSON.parse(localStorage.getItem('users'));
// → 데이터가 없으면 null이 되어 에러 발생

// ✅ 올바른 코드
const users = JSON.parse(localStorage.getItem('users')) || [];

// 에러 3: 폼 제출 시 페이지 새로고침
// ❌ 틀린 코드 (addEventListener 없이)
function handleSubmit() {
    // 폼이 제출되면서 페이지가 새로고침됨
}

// ✅ 올바른 코드
function handleSubmit(event) {
    event.preventDefault(); // 기본 동작 방지
    // 실제 처리 로직
}

✅ 개발 완료 체크리스트

  • 프로젝트 폴더 구조가 체계적으로 구성되어 있다
  • 모든 HTML 페이지가 완성되어 있다

  • localStorage를 활용한 데이터 저장이 정상 작동한다

  • 주요 브라우저에서 테스트를 완료했다
